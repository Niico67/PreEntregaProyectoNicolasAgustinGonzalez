Qué es Linux?

Linux es un sistema operativo de código abierto basado en Unix, diseñado para ser una alternativa libre y gratuita a otros sistemas comerciales como Windows o macOS. Se utiliza en una amplia variedad de dispositivos, desde servidores, supercomputadoras, hasta dispositivos móviles y computadoras personales. A diferencia de otros sistemas, Linux permite a los usuarios modificar su código fuente, lo que lo hace extremadamente flexible y personalizable.

Distribuciones/Distros

Linux no es un sistema operativo único, sino que está compuesto por diferentes distribuciones (o distros), cada una de ellas diseñada con características particulares y adaptada a distintos usos. Entre las más populares se encuentran:

  • Ubuntu: Es una de las distribuciones más amigables para los nuevos usuarios de Linux. Ideal para quienes buscan una experiencia similar a Windows o macOS.
  • Debian: Conocido por su estabilidad, Debian es ideal para servidores y usuarios que priorizan la fiabilidad y seguridad.
  • Fedora: Respaldada por Red Hat, Fedora es una distro que se enfoca en integrar las últimas tecnologías de software libre.
  • Arch Linux: Destinada a usuarios avanzados que desean un control completo sobre su sistema operativo. Su instalación y configuración requieren experiencia técnica.

Glosario de Linux:

Antes de llegar a la sección de comparación, voy a dejar una lista sobre términos con sus descripciones que se usan muy frecuente en el mundo de Linux, para entender un poco más sobre cada distribución y sus diferencias con respecto a otras:

  • Rolling Release: es un modelo de actualización que distribuye software continuamente en lugar de lanzar versiones fijas cada cierto tiempo. Esto significa que el sistema recibe actualizaciones constantes de paquetes, como el kernel o el entorno de escritorio, sin necesidad de reinstalar o actualizar a una nueva versión mayor, manteniendo el sistema siempre actualizado.
  • Estable: Prioriza la confiabilidad sobre las últimas novedades de software, lo que se traduce en un sistema operativo que rara vez falla, con actualizaciones que se enfocan en correcciones de errores y seguridad. Esto se logra generalmente mediante un ciclo de lanzamiento más lento, utilizando software con versiones probadas y limitando la frecuencia de actualizaciones importantes para evitar regresiones.
  • LTS: Siglas de «Long Term Support». Se usa para indicar que un sistema operativo o programa va a estar soportado durante más tiempo, siendo en donde más se ve en las versiones LTS de Ubuntu que salen cada dos años y están soportadas durante 5.
  • Entorno gráfico/Desktop Environment: Es un grupo de componentes que proporcionan los elementos de la interfaz de usuario como iconos, barras de herramientas, fondos de pantalla y widgets. Gracias al entorno gráfico podemos usar Linux gráficamente con un ratón y teclado como lo hacemos en Windows y macOS. Entre los más famosos tenemos GNOME, Plasma o XFCE.
